Projet 1:
PIPOUNES CREATION VERSION BTS - Laissé tel quel

C'est un site que j'ai créé au début du BTS SIO pour m'entraîner à coder un site de A à Z, avec seulement les bases apprises, mais pas complètes.

Langages utilisés :

HTML, CSS.

Points positifs :

- Tous les liens redirigent vers leur page correspondante.
- Il est possible de s'inscrire à la Newsletter du site via un formulaire d'inscription.

Points négatifs :

Projet 2:
TO DO LIST - Terminé

Pour gérer tes tâches du quotidien avec facilité, encouragements et mignonnerie, quoi de mieux ?
Que ce soit pour le travail, les courses, ou même les projets personnels, elle est faite pour toi !

Langages utilisés :

HTML, CSS, JavaScript.

Fonctionnalités :

- Toutes les cases sont modifiables.
